# 2026 FIFA World Cup Analytics Dashboard

I built an interactive data visualization dashboard that can be used for exploring match statistics from the 2026 FIFA World Cup.

This project analyzes match-level data from the ongoing 2026 tournament, allowing users to explore team performance, scoring trends, tournament progression, and other key insights through an interactive interface.

## Features

- Interactive visualizations
- Team performance metrics
- Tournament-wide statistics
- Match explorer with filtering
- Goals and scoring analysis
- Stage-by-stage tournament insights

## Dataset

The dataset contains match-level information for the 2026 FIFA World Cup, including:

- Match date
- Tournament stage
- Teams
- Final scores
- Match winner
- Penalty shootout indicator
- Match notes and notable events

However it is worth mentioning that the tournament is still in progress. Thus the dataset represents a snapshot of the competition and can be updated as new matches are played.

## Tech Stack

- Pandas
- Plotly
- Streamlit
